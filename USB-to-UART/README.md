# USB-to-UART Bridge

A USB to UART Bridge translates data between a USB Port of a computer and a microcontroller’s UART interface. It creates a Virtual COM Port so terminal software like PuTTY can use a microcontroller’s serial TX/RX pins as a standard serial port for debugging, programming and data logging. It manages data flow to ensure the computer does not overwhelm the microcontroller’s slow buffer. It passes the selected communication speed (e.g. 115200 baud) to the bridge hardware so that both ends transmit and receive at the same pace.


# Module Features:

- CP2102 IC – The CP2102 IC integrates a USB transceiver, UART engine, 5V to 3.3V regulator and an internal oscillator. It can be powered from USB or an external 3.3V source and provides up to four GPIO pins that can be reconfigured. 
- Integrated regulator – The CP2102’s internal regulator powers the core and can also supply external circuitry when the module is powered from USB.
- USB Connector – The USB connector provides the physical interface between the  computer and the Printed Circuit Board. It supplies 5V power from the USB host and carries the USB differential data signals required for communication with the CP2102 IC.
- UART Interface – The UART interface exposes the serial communication pins TXD, RXD, GND, and control signals generated by the CP2102. These pins enable communication with external microcontrollers, embedded systems and development boards using standard asynchronous serial protocols.
- Bidirectional TVS Diodes –  The TVS diodes protect the USB data lines from electrostatic discharge (ESD) and voltage transients. They clamp high voltage spikes to safe levels, improving reliability and durability of the USB interface without affecting the signal transmission.
- Capacitors –  The capacitors in the circuit provide power supply decoupling and noise filtering. They stabilise the supply voltage, suppress high-frequency noise and ensure the CP2102 IC operates reliably.
